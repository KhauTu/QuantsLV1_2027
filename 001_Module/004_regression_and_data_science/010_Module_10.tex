\subsection{Linear Regression Analysis}
Regression analysis estimates the relationship between a dependent variable ($Y$), whose variation we want to explain, and an independent variable ($X$), which is hypothesized to influence or explain changes in $Y$. 

In simple linear regression, the population model is expressed as:
\begin{equation}
	Y_i = \beta_0 + \beta_1 X_i + \epsilon_i, \quad i = 1, \dots, n
	\label{eq:pop_regression_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 1.1, Page 524
where:
\begin{itemize}
	\item $Y_i$ is the dependent variable (or response variable) for observation $i$.
	\item $X_i$ is the independent variable (or explanatory variable) for observation $i$.
	\item $\beta_0$ is the intercept coefficient, representing the expected value of $Y$ when $X = 0$.
	\item $\beta_1$ is the slope coefficient, representing the change in the expected value of $Y$ for a one-unit change in $X$.
	\item $\epsilon_i$ is the error term (or disturbance), capturing random noise and variables omitted from the model.
\end{itemize}

The estimated regression line is:
\begin{equation}
	\hat{Y}_i = \hat{\beta}_0 + \hat{\beta}_1 X_i
	\label{eq:est_regression_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 1.1, Page 524
where $\hat{Y}_i$ is the predicted value of the dependent variable for observation $i$, and $\hat{\beta}_0$ and $\hat{\beta}_1$ are the sample estimates of $\beta_0$ and $\beta_1$.

\subsection{Coefficient Estimation of Linear Regression}
The Ordinary Least Squares (OLS) estimation criterion determines the coefficients $\hat{\beta}_0$ and $\hat{\beta}_1$ by minimizing the Sum of Squared Errors (SSE), which is the sum of the squared differences between the observed values $Y_i$ and the predicted values $\hat{Y}_i$:
\begin{equation}
	\text{SSE} = \sum_{i=1}^n (Y_i - \hat{Y}_i)^2 = \sum_{i=1}^n e_i^2
	\label{eq:sse_definition_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 1.2, Page 527

The OLS estimators for the slope and intercept are calculated as:
\begin{equation}
	\hat{\beta}_1 = \frac{\text{Cov}(X, Y)}{\text{Var}(X)} = \frac{\sum_{i=1}^n (X_i - \bar{X})(Y_i - \bar{Y})}{\sum_{i=1}^n (X_i - \bar{X})^2}
	\label{eq:slope_estimator_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 1.2, Page 527
\begin{equation}
	\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1 \bar{X}
	\label{eq:intercept_estimator_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 1.2, Page 527
where $\bar{X}$ and $\bar{Y}$ are the sample means of the independent and dependent variables, respectively.

\subsection{Estimation Residuals}
The difference between the observed value $Y_i$ and the estimated/predicted value $\hat{Y}_i$ is the residual ($e_i$):
\begin{equation}
	e_i = Y_i - \hat{Y}_i
	\label{eq:residual_definition_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 1.3, Page 530
The residuals are the sample counterparts to the population error terms ($\epsilon_i$). Figure~\ref{fig:mod10_exhibit_3} illustrates the residuals plotted around a horizontal line at $e_i = 0$, reflecting the vertical distance of each observation from the regression line.

\subsection{Data Types}
Financial regression models typically utilize three types of data structures:
\begin{itemize}
	\item \textbf{Cross-sectional data:} Observations on multiple entities (e.g., different companies, sectors, or countries) at a single point in time. It is indexed by $i = 1, \dots, n$.
	\item \textbf{Time-series data:} Repeated observations of a single entity (e.g., a country's GDP growth or a company's stock return) across multiple time periods. It is indexed by $t = 1, \dots, T$.
	\item \textbf{Panel data:} A combination of cross-sectional and time-series data, observing multiple entities repeatedly over time.
\end{itemize}

\begin{itemize}

\subsection{Assumptions of the Simple Linear Regression Model}
To ensure that the OLS estimators are Blue (Best Linear Unbiased Estimators) and that statistical inferences are valid, 5 key assumptions must hold:
\begin{enumerate}
	\item \textbf{Linearity:} The relationship between the dependent variable $Y$ and the independent variable $X$ is linear in parameters.
	\item \textbf{Homoskedasticity:} The variance of the error terms ($\epsilon_i$) is constant across all observations: $\text{Var}(\epsilon_i) = \sigma^2$ for all $i$.
	\item \textbf{Independence:} The error terms are independent of one another, implying they are uncorrelated across observations: $\text{Cov}(\epsilon_i, \epsilon_j) = 0$ for all $i \neq j$.
	\item \textbf{Normality:} The error terms ($\epsilon_i$) are normally distributed. This is crucial for small sample hypothesis testing.
	\item \textbf{Exogeneity:} The expected value of the error term conditional on the independent variable is zero: $\text{E}(\epsilon_i | X_i) = 0$.
\end{enumerate}

\subsection{Residual Analysis}
Residual plots (plotting residuals $e_i$ against $X_i$ or time $t$) are crucial diagnostics to identify violations of OLS assumptions:
\begin{itemize}
	\item \textbf{Violation of Linearity:} Residuals show a distinct curved (e.g., U-shaped) pattern when plotted against $X$, suggesting that a linear form is inappropriate. Figure~\ref{fig:mod10_exhibit_5} illustrates this violation.

		\centering
		\includegraphics[width=0.85\linewidth]{002_figs/010_Module_10/10.1-Nonlinear.png}
		\caption{Fitting a Linear and a Non-Linear Model to Non-Linear Data}
		\label{fig:mod10_exhibit_5}
	\end{figure}
	
	\item \textbf{Heteroskedasticity:} The spread of the residuals increases or decreases systematically with $X$ (creating a funnel shape). Non-constant variance invalidates standard error estimates, making $t$-statistics unreliable. Figure~\ref{fig:mod10_exhibit_7} shows a comparison of regressions on data with two variance regimes.
	
	\begin{figure}[!htbp]

		\caption{Regressions on Data with Two Different Regimes and Residuals}
		\label{fig:mod10_exhibit_7}
	\end{figure}
	
	\item \textbf{Autocorrelation (Serial Correlation):} Residuals exhibit a systematic trend or cyclical pattern over time (e.g., positive residuals followed by positive residuals). Common in time-series data, this violates the independence assumption. Figure~\ref{fig:mod10_exhibit_8} compares independent and autocorrelated residuals.

\end{itemize}

\subsection{Evaluating Goodness-of-Fit and Regression Coefficients}
Goodness-of-fit measures how well the independent variable explains variation in the dependent variable. Total variation is decomposed into:
\begin{equation}
	\text{SST} = \text{SSR} + \text{SSE}
	\label{eq:sst_decomposition_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 2.3, Page 540
where:
\begin{itemize}
	\item \textbf{Total Sum of Squares (SST):} Total variation of the dependent variable around its mean: $\text{SST} = \sum (Y_i - \bar{Y})^2$.
	\item \textbf{Regression Sum of Squares (SSR):} Variation explained by the regression line: $\text{SSR} = \sum (\hat{Y}_i - \bar{Y})^2$.
	\item \textbf{Sum of Squared Errors (SSE):} Unexplained variation (residuals): $\text{SSE} = \sum (Y_i - \hat{Y}_i)^2$.
\end{itemize}

The \textbf{Coefficient of Determination ($R^2$)} measures the proportion of total variation in $Y$ explained by the regression model:
\begin{equation}
	R^2 = \frac{\text{SSR}}{\text{SST}} = 1 - \frac{\text{SSE}}{\text{SST}}
	\label{eq:r2_definition_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 2.3, Page 541
In simple linear regression, the correlation coefficient $r$ is related to $R^2$ by:
\begin{equation}
	r = \text{sign}(\hat{\beta}_1) \times \sqrt{R^2}
	\label{eq:correlation_r2_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 2.3, Page 541

The \textbf{Standard Error of the Estimate (SEE)} measures the dispersion of the observed values around the regression line:
\begin{equation}
	s_e = \sqrt{\text{MSE}} = \sqrt{\frac{\text{SSE}}{n - 2}}
	\label{eq:see_definition_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 2.4, Page 544

To test whether the independent variable has a significant linear relationship with the dependent variable, we conduct a hypothesis test on the slope coefficient:
\[ H_0: \beta_1 = 0 \quad \text{vs} \quad H_a: \beta_1 \neq 0 \]
The test statistic follows a $t$-distribution with $n-2$ degrees of freedom:
\begin{equation}
	t = \frac{\hat{\beta}_1 - 0}{s_{\hat{\beta}_1}}
	\label{eq:t_stat_slope_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 2.4, Page 546
where the standard error of the slope estimator $s_{\hat{\beta}_1}$ is:
\begin{equation}
	s_{\hat{\beta}_1} = \frac{s_e}{\sqrt{\sum_{i=1}^n (X_i - \bar{X})^2}}
	\label{eq:se_slope_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 2.4, Page 546

\textbf{Calculations \& Analysis:}
\begin{enumerate}
	\item \textbf{Goodness-of-Fit ($R^2$):}
	\[ R^2 = \frac{\text{SSR}}{\text{SST}} = \frac{0.029}{0.039} = 0.7436 \, (74.36\%) \]
	Interpretation: $74.36\%$ of the monthly excess returns variation of FIRN AG is explained by the market risk premium.
	\item \textbf{Correlation Coefficient ($r$):}
	\[ r = \sqrt{0.7436} = 0.8623 \]
	Since the slope coefficient is positive ($0.8260$), the correlation between FIRN AG and the market is $+0.8623$.
	\item \textbf{Standard Error of the Estimate (SEE):}
	\[ s_e = \sqrt{\text{MSE}} = \sqrt{0.000455} = 0.0213 \]
	\item \textbf{Significance of the Slope:}
	At a $5\%$ significance level, the critical $t$-value with $22$ degrees of freedom is $2.074$.
	Since the calculated $t$-statistic for the slope is $7.988 > 2.074$, we reject the null hypothesis $H_0: \beta_1 = 0$. The slope is statistically significant.
\end{enumerate}

\subsection{Prediction Using a Simple Regression}
For a given value of the independent variable ($X_p$), the predicted value of the dependent variable ($\hat{Y}_p$) is:
\begin{equation}
	\hat{Y}_p = \hat{\beta}_0 + \hat{\beta}_1 X_p
	\label{eq:predicted_value_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 3.1, Page 550

To capture the uncertainty of our prediction, we calculate the \textbf{Prediction Interval} for the dependent variable. The variance of the prediction error ($s_{pred}^2$) is:
\begin{equation}
	s_{pred}^2 = s_e^2 \left[ 1 + \frac{1}{n} + \frac{(X_p - \bar{X})^2}{\sum_{i=1}^n (X_i - \bar{X})^2} \right]
	\label{eq:variance_prediction_error_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 3.1, Page 550

The $1-\alpha$ confidence level prediction interval for the estimated value is:
\begin{equation}
	\hat{Y}_p \pm t_{crit} \times s_{pred}
	\label{eq:prediction_interval_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 3.1, Page 550
where $t_{crit}$ is the critical value of the $t$-distribution with $n-2$ degrees of freedom at the significance level $\alpha/2$.

\begin{examtip}
Notice in Equation~\ref{eq:variance_prediction_error_m10} that the variance of the prediction error increases as $X_p$ moves further away from the sample mean $\bar{X}$. This means our predictions are much more precise when $X_p$ is close to the average of our historical data, and become highly uncertain for extreme inputs!
\end{examtip}

\subsection{Modeling Non-Linear Relationships}
If the relationship between variables is non-linear, we can apply logarithmic transformations to achieve linearity. Table~\ref{tab:mod10_functional_forms} summarizes the three primary logarithmic functional forms.

\begin{table}[!htbp]

2. Sử dụng \textbf{Lin-Log} khi tốc độ tăng trưởng giảm dần theo quy mô (hiệu ứng lợi ích cận biên giảm dần).
3. Sử dụng \textbf{Log-Log} để tìm độ co giãn của cầu theo giá hoặc tìm mức độ nhạy cảm không đổi giữa hai đại lượng tỷ lệ.}}

with $s_{\hat{\beta}_1} = 0.0077$.

\textbf{Analysis:}
\begin{enumerate}
	\item \textbf{Interpretation of the Slope ($\hat{\beta}_1 = 0.2951$):}
	For every 1 CHF increase in revenue per share, the market capitalization increases by approximately $29.51\%$.
	Specifically, the exact relative change is:
	\[ e^{0.2951} - 1 \approx 0.3432 \, (34.32\%) \]
	\item \textbf{Hypothesis Testing:}
	To test if the relationship is statistically significant at the $1\%$ significance level ($t_{crit} \approx 2.75$ with $32$ df):
	\[ t = \frac{0.2951}{0.0077} = 38.32 \]
	Since $38.32 > 2.75$, we reject $H_0$. The slope is highly statistically significant.
\end{enumerate}
\end{tcolorbox}

\textcolor{blue}{\textit{CAPM là ứng dụng kinh điển nhất của hồi quy tuyến tính đơn trong đầu tư tài chính. Phương trình hồi quy CAPM (còn gọi là Security Characteristic Line - SCL) biểu diễn phần bù rủi ro của tài sản ($R_i - R_f$) theo phần bù rủi ro của thị trường ($R_m - R_f$). Hệ số góc của phương trình này chính là hệ số Beta ($\beta_i$), đại diện cho mức độ rủi ro hệ thống của cổ phiếu. Nếu một cổ phiếu có Beta lớn hơn 1, nó sẽ biến động mạnh hơn thị trường và ngược lại.}}

\subsection{CAPM Regression Framework}
The CAPM asserts that the expected excess return on an asset is linearly related to the market risk premium. We express it as a simple linear regression:
\begin{equation}
	R_{i,t} - R_{f,t} = \alpha_i + \beta_i (R_{m,t} - R_{f,t}) + \epsilon_{i,t}
	\label{eq:capm_regression_m10}
\end{equation}
% Reference: CFA Curriculum 2027, Vol 1, Module 10: Applications of Simple Linear Regression in Finance, Section 4.1, Page 566
where:
\begin{itemize}
	\item $R_{i,t} - R_{f,t}$ is the excess return of asset $i$ over the risk-free rate at time $t$ (dependent variable $Y$).
	\item $R_{m,t} - R_{f,t}$ is the market risk premium at time $t$ (independent variable $X$).
	\item $\beta_i$ is the systematic risk (or market beta) of the asset, reflecting its sensitivity to market movements.
	\item $\alpha_i$ is the intercept, representing the abnormal return (Jensen's alpha). Under market efficiency, the expected value of $\alpha_i$ is zero.
\end{itemize}

	\item \textbf{Statistical Significance of Beta:}
	The calculated $t$-statistic for the slope coefficient is $16.243$. Compared to the critical $t$-value at the $5\%$ level of significance ($t_{crit} \approx 1.96$), $16.243 > 1.96$. Thus, LVMH's beta is statistically significant.

	\item \textbf{Intercept Alpha ($\hat{\alpha} = 0.0010$):}
	The intercept represents the abnormal excess return of LVMH. The calculated $t$-statistic is $0.525 < 1.96$, showing that the alpha is not statistically different from zero, which is consistent with the CAPM.
\end{enumerate}
